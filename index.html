Validation & Accessibility Checklist

  1. Validate HTML at the W3C validator: validator.w3.org. Save a screenshot of the "Document checking completed. No warnings or errors." result.
  2. Validate CSS at the W3C CSS validator: jigsaw.w3.org/css-validator. Save a screenshot of the successful validation.
  3. Run the WAVE accessibility tool on the published GitHub Pages URL: wave.webaim.org. Save a screenshot of the WAVE results page and upload it to your repository.
  4. Fix any errors the validators or WAVE report and repeat until you have error-free validation screenshots (or no major WAVE errors).

Place the saved screenshots (HTML validator, CSS validator, WAVE) into your GitHub repo `project03` before final submission.
